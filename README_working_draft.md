# IDEN 0.1.0


###### NOTE: This markdown document is formatted for "peerpub" and uses "iden://" URI links as well as inline expansions which will not work or render on GitHub.

Iden is a lightweight foundation for distributed, coordinated services built on self-generated, perpetually maintainable identities.

## The Basics:  
  
* ### Iden Token:
	
	An iden is a 32-byte unique identifier with 2³² secret indexed *states* with each state at idx =*n* resolving to the state at index *n-1*. This provides a fast, non-interactive proof of continuity, with each newly revealed state authenticating ownership and updating the system.
	
* ### States:
	A *state* is 36 bytes: a 32-byte hash and a 4-byte `u32` index in little-endian order.  Each state at index *n* is computed by hashing the state at index *n+1*, and appending the decremented index. 
	
	An iden is generated by choosing a random 32-byte *seed state* (regeneration key) and stepping down from a starting idx set to `0xFFFFFFFF` (decimal 4,294,967,295) to idx 0 (with the final hash being the iden). The seed state (and hopefully some intermediate checkpoints) can be saved to an *iden pad* file.
	
	Importantly, the seed state can - *and should* - be stored 'offline', ideally as a paper record.  It is unnecessary and ill-advised to trust any device with any high state beyond what will be needed in a 'reasonable' period of time. Further, the state stored on any device *should* always be encrypted.
	
* ### String Representation:

	Both states and strings are prefixed by an IPFS 'Multibase' character 'z' to
denote base58btc and the binary state before encoding is prefixed by a single byte
version number(present for future-proofing).

	Python encoder and decoder:

* ### Signal Store & Messages:

	An iden node consists of a *signal store* governed by a *message processor*.  
There are three core message types:
	
	* **Report**:
	
		['re'][32 byte iden][36 byte state]
		
	* **Claim**:
	 
		['cl'][32 byte iden][32 byte mix][4 byte idx]
		
	* **Proof**:
	
		['pr'][32 byte iden][36 byte state][64 byte signal]
		
		
	The signal store retains a record of the highest known state for an iden and a limited buffer of the most recent signals registered to any states if present (which may or may not include the current high state).  
	  

	## Node Rules:
	
	**Claims:** A node will cache any new claim promising a higher idx than the known high state for an iden. The implementation here ignores claims for unknown idens until a state has been established on record, which is fine and probably advisable. Any time a new report or proof is accepted, any 'open claims' are discarded for lower idx values.
	
	**Reports:** If a reported state is higher than the current record — and the number of steps to verify convergence is within safe bounds (to protect against DoS) — the node will check if the new state steps down to the one on record. If so, and if the old high-state has no associated signal, the new state will replace the top record. If there was a signal associated with the prior high-state, the new state will be appended to the top of the record.
	
	  
	 **Proofs:** As with reports, a proof that reveals a new high state will be accepted if the state steps down properly. In addition, if the hash of `[signal | state]` matches the expected *mix* from a previously cached claim promised for that idx, then the proof is accepted and both state and signal are registered.
	 
	 
	 	At the level of the basic signal network, you can think of your iden 
	 	as the public address of a pipe with a history buffer that anyone can 
	 	read from but that only you can write to.


	## Building Services:
	
	The signal cache is a store that can be read quickly to access any recent signals registered to an iden. The signal is a sixty-four byte record space that may hold any information needed by any service to coordinate further action. In the *basenet* service, for instance, one or two payload keys can be registered in a signal (and up to 64 variations on signals which may contain the same keys).  

   		Note that all messages we've discussed so far are one-way — no 	
		acknowledgments or replies are needed. They're also compact (70 to 134 
		bytes), making them UDP-friendly.
  
	When a basenet node receives an *offer* message which offers for delivery content specified by iden, idx, and hashkey, it reads the signal (if any), doubles the signal field into a 128-byte array, and checks to see if the key is found in the resulting buffer (assuming it doesn't already have the record on file). If that record is not yet on file and the key (and hash of the payload) matches, then the record is immediately accepted.
	

	
	## Basenet:
	
	Basenet is a simple URI addressable data-store and 'feed':

 	Data Frames:

	* Data-frames are limited to no more than sixty-four kilobytes in size and must be valid UTF-8 encoded data. They are required to begin with a dictionary which must contain at least the key '_' (that is a single underscore) with the value '0' or '1', designating the frame type:

		* Type 0 frames replace previous type 0 records.
	
		* Type 1 frames add to the 'top' of a feed of frames with at least 
		128 kb of storage to be allowed by the node.
	
	The header must end with a document divider '---', and may be followed by any valid UTF-8 encoded data.

	'_' is the only required field, any additional fields added to the header will not be available as path fields for retrieval (see below) and will generally be ignored. If you want to sign the payload frame, this is where you would put your signature.

	### Basenet URI Retrieval:

	Content is referenced using the 'iden://' protocol specifier:

	#### The most recently added type 1 (feed) frame:

	\iden://z1HRUsTNcYMkN5WPm9s1YjGaLUVs58RVRHPjBrV1kYwdAJ
	
	#### The current type 0 (pinned) frame:

	\iden://z1HRUsTNcYMkN5WPm9s1YjGaLUVs58RVRHPjBrV1kYwdAJ.0
	
	#### The type 1 frame at idx = 1,000,000:

	\iden://z1HRUsTNcYMkN5WPm9s1YjGaLUVs58RVRHPjBrV1kYwdAJ.1000000
	

	#### If the main document is a YAML dictionary:
	
	\iden://z1HRUsTNcYMkN5WPm9s1YjGaLUVs58RVRHPjBrV1kYwdAJ.42/files/src/README.txt
	
	(This should fail gracefully, returning the structure as far down 
	the parse path as it is able to reach.)


## Implementation And Tools:

This implementation consists mostly of a small multi-function Linux/Unix executable written in Rust which provides essential user tools and a set of micro-services comprising the base system (signaling and basenet with resolved full-frame retrieval and basic TCP connectivity).  It features configurable throttling, sharding, message, and claim caches as well as script-able startup and shutdown of services along with REPL control.

A web front-end, YAML resolution, and *pubnet* (described below) as well as a library for user tools are provided by a single Python "basenet" script.

### Getting Started:

To initialize a new node store:
	
	cd; iden init

To generate a new pad:

	iden generate <output-file>
	
To store an encrypted checkpoint:

	iden store <input-pad> <name> <idx>
	
Launch the pad manager:

	iden padman <name>
	
		manages the pad .iden/name.pad, serving on states on the socket .iden/padman

### Ed25519_IPv4_TCP_0.1.0 API Specification

This document describes the TCP input interface for the IDEN signaling system.

---

## Peers

**Command:**  
`'pe' + [u16_le: count]`

**Response:**  
`[u16_le: record_count] + [record_len_1][record_1] + [record_len_2][record_2] + ...`

Each record is:  
`[32-byte iden] + [UTF-8 string, null-terminated]`

---

## Handshake

**Command:**  
`'hi' + [64-byte challenge]`

**Response:**  
`[32-byte pubkey] + [64-byte signature] + [64-byte challenge]`

**Follow-up from peer:**  
`[32-byte pubkey] + [64-byte signature] + [optional UTF-8 string, null-terminated]`

**Final Response:**  
Observed IP and port, UTF-8 string.

---

## Message Count in Last `t` Seconds

**Command:**  
`'ct' + [f64_le: t]`

**Response:**  
`[u32_le: count]`

---

## Get All Messages in Last `t` Seconds

**Command:**  
`'gt' + [f64_le: t]`

**Response:**  
`[u32_le: total_length] + concatenated messages`

Messages are streamed in chunks of 1380 bytes.

---

## Dedicate Link

**Command:**  
`'de' + [32-byte iden] + [36-byte state]`  
*(70 bytes total)*

**Response:**  
Currently none — should be updated to return mproc’s response byte:  
- `0 = OK`
- `1 = Bad size`
- `2 = Too low`
- `3 = Too far`
- `4 = False`

---

## Index

**Command:**  
`'ix' + [32-byte iden]`

**Response:**  
- `[u32_le: idx]` if found  
- Single `0` byte if not found

---

## State

**Command:**  
`'st' + [32-byte iden]`

**Response:**  
- `[36-byte state]` if found  
- Single `0` byte if not found

---

## Report

**Command:**  
`'re' + [32-byte iden] + [36-byte state]`  
*(70 bytes total)*

**Response Codes:**  
- `0 = OK`  
- `1 = Bad size`  
- `2 = Too low`  
- `3 = Too far`  
- `4 = False`

---

## Claim

**Command:**  
`'cl' + [32-byte iden] + [32-byte mix] + [u32_le: idx]`  
*(70 bytes total)*

**Response:**  
None

---

## Proof

**Command:**  
`'pr' + [32-byte iden] + [36-byte state] + [64-byte signal]`  
*(134 bytes total)*

**Response Codes:**  
- `0 = OK`  
- `1 = Bad size`  
- `2 = Too low`  
- `3 = Too far`  
- `4 = False`

---

## Version

**Command:**  
`'ve'`

**Response:**  
UTF-8 encoded version string

def crypt_state(state,password):
    ''' Encrypt (or decrypt) a state by xor of password hash. '''
    from hashlib import sha256
    if type(state) == str:
        state = unpack_state(state)
    p = sha256( password.encode(encoding='utf-8') ).digest()
    s = state[:-4]; x = bytes(a ^ b for (a,b) in zip(s, p))
    return(x+state[-4:])

def pack_state(state):
    ''' Return string represtentation for a state. '''
    from leb128 import i as varint
    import base58
    if type(state) == str:
        return(state)
    return((b'z'+base58.b58encode(bytes(varint.encode(0)+state))).decode())

def unpack_state(s):
    ''' Return a binary state given a string representation. '''
    import base58
    if type(s) == bytes:
        return(s)
    s = s[1:]; s = base58.b58decode(s)
    assert s[0] == 0
    s = s[1:]
    return(s)

def state_idx(state):
    ''' Returns integer state index for a state. '''
    if type(state) == str:
        state = unpack_state(state)
    return(int.from_bytes(state[-4:],'little',signed=False))

def step(state, start=False):
    ''' Advance a state one generation step.'''
    from hashlib import sha256
    a = sha256(state).digest(); b = int.from_bytes(state[-4:],'little')
    return(a+(b-1).to_bytes(4,'little',signed=False))

class Frame:
    def __init__(self, content: dict = None):
        self.body = content or {}

    def add_file(self, path: str, data: str):
        parts = path.strip("/").split("/")
        node = self.body.setdefault('files', {})
        for part in parts[:-1]:
            node = node.setdefault(part, {})
        node[parts[-1]] = data

    def author(self, name: str):
        self.body['author'] = name

    def geoloc(self, lat: float, lon: float):
        self.body['geoloc'] = {'lat':lat,'lon':lon}

    def lang(self, lang: str):
        self.body['lang'] = lang

    def title(self, title: str):
        self.body['title'] = title

    def link_file(self, path: str, uri: str, hash: str = None):
        link = {'link': uri}
        if hash is not None:
            link['hash'] = hash
        self.add_file(path, link)

    def time(self, timestamp: float = None):
        import time
        self.body['time'] = timestamp or time.time()

    def to_bytes(self):
        return yaml.dump(self.body).encode()


def Publish(body,
            page: int = 1,
            r: bytes = bytes([0] * 32),
            host: str = "127.0.0.1",
            signal_port: int = 4004,
            basenet_port: int = 4040):

    def Hash(b):
        return sha256(b).digest()

    def Mix(state, signal):
        assert isinstance(signal, bytes) and len(signal) == 64
        return Hash(state + signal)

    def offer_msg(iden, n, key):
        assert len(iden) == 32 and len(key) == 32
        return b'of' + iden + n.to_bytes(4, 'little') + key

    def tcp_send(data, port):
        try:
            with socket.socket(socket.AF_INET, socket.SOCK_STREAM) as s:
                s.connect((host, port))
                s.sendall(data)
                buff = []
                response = s.recv(1380)
                while response:
                    buff.append(response)
                    response = s.recv(1380)
            return b''.join(buff)
        except Exception as e:
            print("Error:", e)
            return b""

    def unix_send(socket_path: str, data: bytes) -> bytes:
        try:
            with socket.socket(socket.AF_UNIX, socket.SOCK_STREAM) as client:
                client.connect(socket_path)
                client.sendall(data)
                return client.recv(65536)
        except Exception as e:
            print(f"Error communicating with unix socket at {socket_path}: {e}")
            return b''

    def tcp_deliver(offer, data):
        try:
            with socket.socket(socket.AF_INET, socket.SOCK_STREAM) as s:
                s.connect((host, basenet_port))
                s.sendall(offer)
                resp = s.recv(2)
                print("Response from basenet:", resp)
                if resp == b"ok":
                    length = len(data).to_bytes(2, 'little')
                    s.sendall(length + data)
                    print("Payload sent.")
                else:
                    print("Did not receive OK. Aborting.")
        except Exception as e:
            print("Delivery error:", e)

    padman = os.path.expanduser('~/.iden/padman')
    iden = unpack_state(unix_send(padman, b'id').decode())
    state = unix_send(padman, b'st')
    idx = state_idx(state)

    if hasattr(body, 'to_bytes'):
        body = body.to_bytes()

    header = yaml.dump({'_': page}).encode()
    full_payload = header + b"---\n" + body

    datahash = Hash(full_payload)
    signal = datahash + r
    mix = Mix(state, signal)

    cl = b'cl' + iden + mix + idx.to_bytes(4, 'little')
    pr = b'pr' + iden + state + signal

    print("Sending claim...")
    print(tcp_send(cl, signal_port))
    time.sleep(0.01)

    print("Sending proof...")
    print(tcp_send(pr, signal_port))

    offer = offer_msg(iden, idx, datahash)
    tcp_deliver(offer, full_payload)

    return offer, full_payload

def rewrite_iden_links(md_text: str, prefix: str = "127.0.0.1:4040/") -> str:
    def replacer(match):
        full = match.group(0)
        if full.startswith("\\iden://"):
            return "iden://" + full[7:]  # remove escape
        return prefix + full
    pattern = re.compile(r'(\\)?iden://[^\s)>\]"\']+')
    return pattern.sub(replacer, md_text)


@app.get("/{path:path}")
async def handle_request(path: str, request: Request):
    if not path.startswith("iden://"):
        return PlainTextResponse("Invalid request format", status_code=400)

    trimmed = path[len("iden://"):]
    parts = trimmed.split("/")
    uri_part = "iden://" + parts.pop(0)
    subkeys = [part.strip() for part in parts if part.strip()]
    response = tcp_send(uri_part.encode())

    if subkeys:
        try:
            yml = yaml.safe_load_all(response)
            next(yml)  # skip header
            yml = next(yml)

            if subkeys[0] == 'pub' and 'pub' not in yml:
                if 'text.md' in yml:
                    return render_markdown_as_html(yml['text.md'], title="PeerPub Document")

            for key in subkeys:
                if isinstance(yml, dict) and key in yml:
                    yml = yml[key]
                    if not isinstance(yml, dict):
                        return PlainTextResponse(yaml.dump(yml))

            return PlainTextResponse(yaml.dump(yml))
        except Exception as e:
            return PlainTextResponse(f"[Error parsing content: {e}]")

    return PlainTextResponse(response)

def expand_inline_iden_blocks(md_text: str) -> str:
    pattern = re.compile(r"(?<!\\):::\s*(iden://[^\s:]+(?:\.[0-9]+)?(?:/[^\s:]+)*)\s*:::", re.IGNORECASE)

    def expand(match):
        full_uri = match.group(1).strip()
        print("Expanding inline block:", full_uri)

        # Split base URI and subpath
        trimmed = full_uri[len("iden://"):]
        parts = trimmed.split("/")
        base_uri = "iden://" + parts[0]
        subkeys = [p.strip() for p in parts[1:] if p.strip()]

        try:
            response = tcp_send(base_uri.encode())

            try:
                text = response.decode("utf-8")
            except:
                return "[[binary content]]"

            try:
                yml = yaml.safe_load_all(text)
                next(yml)  # Skip header
                content = next(yml)

                for key in subkeys:
                    if isinstance(content, dict) and key in content:
                        content = content[key]
                    else:
                        break

                return content if isinstance(content, str) else repr(content)

            except:
                return text  # Couldn't parse as YAML — return as-is

        except:
            return ""  # No errors returned, just nothing

    return pattern.sub(expand, md_text)
